# ROS Realtime API
Sample code to call the ros2 apis

## Setup
There is one time setup that needs to be done by a standard bots engineer.

## Building

```
./build.sh
```

## Running

```
./run.sh
```

## Run in docker environement

1. `./run_shell.sh`
1. `cd /src`
1. `python3 ./src/plan_and_execute.py`
